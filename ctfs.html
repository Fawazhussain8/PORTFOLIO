Shadow Ledger

SOC-002 · Hard
Valora Retail
SOC Hard

A mid-sized e-commerce company detects signs of a network breach with a 10-minute response window. Participants pivot from an anomalous SMB session in network logs to uncover an accessed share, decode a loader payload from event logs, identify a rogue Active Directory account, terminate its malicious process, and block the C2 IP — simulating a real-time SOC containment scenario.

SMB Analysis Event Log Investigation Active Directory C2 Blocking T1078 · Valid Accounts

Design Process

01
Threat Scenario Modelling
Each challenge begins with a real-world attack technique or investigation scenario mapped to MITRE ATT&CK.
02
Artifact Engineering
Building challenge files — PCAPs, disk images, PDFs, binaries — with embedded flags and deliberate forensic trails.
03
Difficulty Calibration
Testing solve paths, setting hint tiers, and balancing challenge depth against intended skill level.
04
Writeup & Documentation
Every deployed challenge includes a solution writeup for post-event learning and analyst debriefing.
